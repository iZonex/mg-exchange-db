=== Accessing ExchangeDB ===

1. HTTP REST API & Web Console (port {{ .Values.service.ports.http }}):

   kubectl port-forward svc/{{ include "exchangedb.fullname" . }} {{ .Values.service.ports.http }}:{{ .Values.service.ports.http }}

   Then open: http://localhost:{{ .Values.service.ports.http }}

2. PostgreSQL Wire Protocol (port {{ .Values.service.ports.postgresql }}):

   kubectl port-forward svc/{{ include "exchangedb.fullname" . }} {{ .Values.service.ports.postgresql }}:{{ .Values.service.ports.postgresql }}

   Connect with: psql -h localhost -p {{ .Values.service.ports.postgresql }}

3. ILP Ingestion (port {{ .Values.service.ports.ilp }}):

   kubectl port-forward svc/{{ include "exchangedb.fullname" . }} {{ .Values.service.ports.ilp }}:{{ .Values.service.ports.ilp }}

=== Health Check ===

   kubectl exec -it {{ include "exchangedb.fullname" . }}-0 -- curl -s http://localhost:{{ .Values.service.ports.http }}/health
